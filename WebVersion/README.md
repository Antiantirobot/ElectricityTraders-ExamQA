# 电力交易题库刷题

一个纯前端中文刷题网站（手机/桌面响应式），直接打开 `index.html` 即可使用。  
当前已接入 Word 文档全量题库导入结果（默认加载 `question_bank.js`）。

## 运行方式

1. 进入目录：`d:\PythonFiles\Testpy`
2. 双击打开 `index.html`（或用浏览器打开）

## 服务器部署（支持手机/电脑进度同步）

### 本地测试

1. 在项目目录执行：
`python sync_server.py`
2. 浏览器打开：
`http://127.0.0.1:8000`

### 服务器部署

1. 将整个目录上传到服务器（至少保留：`index.html`、`style.css`、`app.js`、`question_bank.js`、`sync_server.py`）
2. 服务器安装 Python 3.10+（无第三方依赖）
3. 启动：
`HOST=0.0.0.0 PORT=8000 python sync_server.py`
4. 通过 Nginx/Caddy 反向代理到 80/443，并配置 HTTPS

## 跨设备同步使用

1. 电脑和手机都访问同一个部署地址（同一域名）
2. 首页“云同步（跨设备）”里填写：
- 服务器地址：同域可留空；跨域填 `https://你的域名`
- 学习账号：例如 `zhangsan`
- 同步密码：自定义
3. 点击“保存配置”
4. 先点击“从云端拉取”（首次无数据时再“上传到云端”）
5. 后续做题会自动同步（可关闭自动同步，改手动拉取/上传）

## 全量题库转换（Word -> CSV/JSON/JS）

已提供脚本：`scripts/convert_docx_to_bank.py`

执行：

`python scripts/convert_docx_to_bank.py`

会生成：

- `question_bank.csv`
- `question_bank.json`
- `question_bank.js`（网页默认加载）

## 已实现功能

- 首页仪表盘：总题量、已练习题数、正确率、最近练习、错题数量
- 章节+题型刷题：顺序/随机、即时判分、上一题/下一题、收藏、错题、查看答案/解析
- 模拟考试：单选100、多选30、判断40，120分钟倒计时，交卷评分、错题自动入本、历史记录
- 错题本：按章节/题型筛选、只看未掌握、标记掌握、移出错题本、重刷
- 收藏夹：按章节/题型筛选与管理
- 学习统计：累计数据、章节掌握、近7天练习趋势、考试成绩趋势
- 题库管理：预留批量导入结构（粘贴文本解析+导入）+ 一键重置为全量题库

## 数据存储

- 使用浏览器 `localStorage` 保存：题库、练习记录、考试记录、错题本、收藏夹
- 清空浏览器站点数据会清除学习记录

## 题库导入预留格式

管理页支持粘贴文本，每行一题：

`章节,题型,题干,选项A,选项B,选项C,选项D,答案,解析`

示例：

`第一章 电力市场基础,single,电力市场的核心特征是？,计划分配,市场定价,取消结算,单一主体定价,B,市场化机制形成价格`

多选题答案可用 `A|C`。
